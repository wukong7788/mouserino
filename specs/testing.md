# Testing Checklist - Mouserino

## 1. 启动与权限
1. 启动应用，确认界面可正常加载。
2. 在未授权 Accessibility 时，确认有明确提示且应用不崩溃。
3. 点击请求权限后，确认可跳转或引导到系统设置。

## 2. 映射行为
1. 开启 remap，总体确认 side back/forward/middle 三个按键可触发配置动作。
2. 将某按键设置为 `No Action`，确认该按键不会触发额外动作。
3. 关闭 remap，确认不再执行自定义映射。

## 3. 持久化
1. 修改多个按键动作和总开关后重启应用。
2. 验证所有配置值保持一致。

## 4. 回归
1. 切换授权状态（授权后再撤销）验证应用稳定性。
2. 重复开关 remap 多次，确认无明显延迟累积或行为异常。
